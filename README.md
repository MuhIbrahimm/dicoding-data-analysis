# Bike Sharing Dashboard
by Muhammad Ibrahim

## Setup Environment 
```
git init
git clone https://github.com/MuhIbrahimm/dicoding-data-analysis
cd dicoding-data-analysis
pip install -r requirements.txt
```

## Run Streamlit
```
streamlit run dashboard.py
```
